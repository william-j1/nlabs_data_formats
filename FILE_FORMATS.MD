# Data Formats

NLABS.app permits file imports for data analysis and rendering. This repository lists the file types currently supported.

# File Types

| Type       | Result       | Notes                    |
|:----------:|--------------|--------------------------|
| BMP        | Image        | ........................ |
| DCM	       | Image        | MONOCHROME1, MONOCHROME2 |
| JPG        | Image        | ........................ |
| PLY        | Shape        | ........................ |
| PNG        | Image        | ........................ |

# Unsupported

If a file type is not supported the data is imported as a row vector.
